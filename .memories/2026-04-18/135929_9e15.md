## Reduced distributed Julie skill set

### WHAT
Reduced the shipped plugin skill set from 11 to 4 and updated distribution metadata to match.

- Kept and shipped: `editing`, `explore-area`, `impact-analysis`, `web-research`
- Removed shipped copies of: `architecture`, `call-trace`, `dependency-graph`, `logic-flow`, `metrics`, `search-debug`, `type-flow`
- Rewrote the shipped `editing` skill around `rewrite_symbol`
- Updated `README.md` tool and skill counts
- Updated `.github/workflows/update-binaries.yml` to copy only the 4 public skills and expect the new count

### WHY
The plugin was still advertising removed tools and shipping stale skills. That created drift between julie source, plugin distribution, and the real MCP surface users see.

### HOW
Deleted the cut skill copies, mirrored the updated public `editing` skill from julie, and aligned the release workflow and README with the reduced distributed set.

### IMPACT
The plugin now ships the intended public skill set and the automated sync workflow will stop reintroducing deleted skills on the next release.
